Приложението има за цел да пресмята информация за покупки направени с карта, като има 3 вида карти (Bronze, Silver, Gold),като всяка е с различна отстъпка.

Примерен вход:

Create Bronze Ivan 500 500
Create Silver Gosho 600 600
Create Gold Stamat 500 600

Checkout Ivan
Checkout Stamat

Изход:

Owner name: Ivan
Purchase Value: $500,00
Discount rate: 2,50%
Discount: $12,50
Total: $487,50

Owner name: Stamat
Purchase Value: $600,00
Discount rate: 7,00%
Discount: $42,00
Total: $558,00
